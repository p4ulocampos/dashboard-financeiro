# 📊 Dashboard Financeiro

Um aplicativo web inteligente para controlar todas as suas finanças de forma simples e visual. Veja quanto você gasta, onde seu dinheiro vai e organize seu orçamento em um único lugar!

---

## 🎯 Para que serve?

Imagine ter um "assistente financeiro" que:
- **Mostra quanto você gastou** no mês em gráficos bonitos
- **Organiza seus gastos** por categoria (alimentação, transporte, lazer, etc.)
- **Rastreia sua receita** (salário, bônus, pensão)
- **Calcula automaticamente** quanto sobra no final do mês
- **Tudo acessível** por um navegador de internet, de qualquer lugar

---

## 📱 Como funciona?

### **Página 1: Dashboard (Painel de Controle)**
Este é o "painel de instrumentos" do seu dinheiro. Aqui você vê:

- **4 Números Principais:**
  - 💰 **Gasto Total**: Quanto você gastou neste mês
  - 📊 **Nº de Lançamentos**: Quantas despesas você registrou
  - 💵 **Receitas Totais**: Quanto você recebeu
  - ✅ **Lucro**: Receitas menos Despesas (o que sobrou)

- **Gráfico de Gastos por Categoria**: Uma barra mostrando qual categoria consome mais dinheiro
- **Tabela de Detalhes**: Lista completa de todas as suas transações

### **Página 2: Realizar Lançamento**
É aqui que você **registra um novo gasto ou receita** no sistema. Basta preencher:
- 📅 **Data**: Quando aconteceu
- 🏷️ **Categoria**: Tipo de gasto (Alimentação, Transporte, etc.)
- 💵 **Valor**: Quanto custou
- 📝 **Descrição**: Anotação (ex: "Supermercado")
- 👤 **Nome do Titular**: Quem gastou
- 🗓️ **Dia de Vencimento**: Quando precisa pagar (se for débito/crédito)
- 🏦 **Banco/Cartão**: Onde vem o dinheiro (Nubank, Itaú, Mercado Pago)

---

## 🎮 Como Usar?

### **Navegação (Menu Lateral Esquerdo)**

1. **Selecione a Página** - Escolha entre Dashboard ou Lançamento
2. **Filtre por Mês e Ano** - Use os seletores para ver dados de um período específico
3. **Filtros Avançados** - Clique em "🛠️ Filtros Avançados" para refinar ainda mais:
   - Por Tipo (Receita ou Despesa)
   - Por Categoria (Alimentação, Transporte, etc.)
   - Por Origem (Cartão de Crédito, PIX, etc.)
   - Por Valor (de quanto a quanto)

### **Botões Principais**

- 🔄 **Atualizar Dados**: Clique para ver os dados mais recentes
- **Registrar no Banco**: Salva um novo lançamento no banco de dados

---

## 📊 Tipos de Registros

O sistema organiza seu dinheiro em 3 categorias principais:

| Tipo | Descrição | Exemplo |
|------|-----------|---------|
| **Receitas** | Dinheiro que entra | Salário, bônus, pensão, férias |
| **Despesas (Cartão de Crédito)** | Gastos com cartão de crédito | Compras no supermercado, combustível |
| **Despesas Fixas** | Contas que repetem todo mês | Aluguel, água, luz, internet |

---

## 🔐 Segurança

- O formulário de lançamentos é **protegido por senha**
- Apenas quem conhece a senha pode adicionar novos gastos
- Seus dados estão armazenados de forma segura na nuvem

---

## 💡 Dicas Úteis

✅ **Atualize frequentemente**: Quanto mais dados você registrar, melhor será o controle
✅ **Use categorias padronizadas**: Facilita na hora de gerar gráficos
✅ **Revise mensalmente**: Veja o Dashboard toda semana para acompanhar
✅ **Use filtros**: Explore dados por período, categoria e origem

---

## 🛠️ Tecnologia (Para Curiosos)

Este projeto usa tecnologias modernas para ser rápido e seguro:
- **Streamlit**: Interface web bonita e responsiva
- **Supabase**: Banco de dados na nuvem (tipo um Google Sheets inteligente)
- **Pandas**: Processa e organiza os dados
- **Plotly**: Cria gráficos interativos e profissionais
